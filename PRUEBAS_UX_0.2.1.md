# Pruebas UX 0.2.1

## Movimientos

- [ ] La fila de una transferencia no muestra una etiqueta azul estrecha o repetida.
- [ ] La ruta de origen a destino se lee sin ocupar varias líneas innecesarias.
- [ ] Pulsar la fila abre `Detalle del movimiento`.
- [ ] El detalle muestra cuenta de origen y destino.
- [ ] El lápiz abre el formulario de edición.
- [ ] El gesto lateral derecho sigue permitiendo eliminar.

## Objetivos

- [ ] Pulsar toda la tarjeta del objetivo destacado abre su detalle.
- [ ] `Ver detalle` abre la misma ficha.
- [ ] La ficha muestra progreso, importes, seguimiento y detalles.
- [ ] El botón `+` permite crear otro objetivo.
- [ ] El lápiz permite editar el objetivo actual.
- [ ] Desde Ajustes > Objetivos, pulsar una fila abre detalle y no edición directa.

## Gráfico

- [ ] La curva no supera el límite superior o lateral del área de trazado.
- [ ] Los puntos inicial y final conservan margen respecto al borde.
- [ ] La curva sigue siendo legible en todos los periodos.
